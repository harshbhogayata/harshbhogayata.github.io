Network Traffic Monitor

Python Scapy Wireshark

Figure 1: Dashboard Interface

Overview

This project was designed to analyze local network traffic in real-time, identifying potential anomalies and visualizing data flow. It serves as a lightweight IDS (Intrusion Detection System) for home networks.

Key Features

  • Real-time packet capturing using Scapy.
  • Protocol distribution analysis (TCP/UDP/ICMP).
  • Alert system for suspicious port scanning activities.
  • Log export to CSV for further analysis.

Technical Implementation

The core logic revolves around a packet sniffer loop. Here is a snippet of the capture logic: 

def packet_callback(packet):
    if packet.haslayer(IP):
        src_ip = packet[IP].src
        dst_ip = packet[IP].dst
        protocol = packet[IP].proto
        
        # Check for suspicious activity
        if protocol == 6: # TCP
            check_port_scan(src_ip, packet[TCP].dport)

        log_packet(src_ip, dst_ip, protocol)

Challenges & Solutions

Performance: Initially, the packet processing was blocking the main thread. I implemented Python's threading module to handle analysis asynchronously, ensuring no packets were dropped during high traffic.

Future Improvements

I plan to add a web-based dashboard using Flask to visualize the traffic graphs in real-time, replacing the current CLI output.
